# 🧠 Cortex  
**AI-powered Commercial Intelligence for Life Sciences**

---

## 🚀 Overview

Cortex is an enterprise AI assistant that transforms **business questions into governed, explainable insights** across commercial data assets.

It enables teams to move from:

**Question → Relevant Data → Analysis → Insight**

— without needing to navigate datasets, dashboards, or write SQL.

Cortex is built on a **metadata-first architecture**, ensuring that every insight is:
- grounded in trusted enterprise data  
- explainable and traceable  
- aligned with governance and access policies  

---

## 🎯 Problem

Commercial teams operate in complex data ecosystems — pricing systems, contracts, claims, CRM, dashboards, and documents.

Despite having data, teams struggle with:

- Finding the right dataset or dashboard  
- Inconsistent KPI definitions across teams  
- Slow, manual data discovery and validation  
- Dependence on data/BI teams for ad-hoc queries  
- Limited trust in black-box AI outputs  

➡️ Analysts often spend more time **finding data** than **analyzing it**

---

## 💡 Solution

Cortex acts as a **commercial intelligence layer** that:

1. Understands business questions in natural language  
2. Discovers relevant data assets using metadata  
3. Validates access and governance  
4. Generates deterministic analysis (SQL / retrieval)  
5. Returns **explainable, business-ready insights**

## ⚙️ Core Capabilities

## 🏗️ Architecture (Conceptual)

Cortex uses a **multi-agent architecture**:

- **Supervisor Agent** → Orchestrates flow and intent parsing  
- **SQL Agent** → Generates & executes structured queries  
- **Document Agent** → Handles file-based intelligence  
- **Dashboard Agent** → Interprets BI assets  
- **Goalkeeper Agent** → Enforces access and governance  

All powered by a unified metadata layer:

👉 **Magic Meta Store**
