# Pharma Action Builder

Pharma Action Builder is a pharma commercial intelligence platform designed to connect machine learning-driven insights with field execution.

It helps commercial teams move from generic engagement plans to personalised **Next Best Action (NBA)** recommendations for every HCP, while enabling analytics teams to build, monitor, and deploy models through a no-code AutoML environment.

## Overview

The platform is built around two core modules:

- **HCP Engagement Console (HEC)** – a field-facing dashboard that provides HCP-specific engagement plans, tactic tracking, performance visibility, and territory intelligence.
- **Analytics Workbench (AWB)** – a no-code AutoML workspace where analytics teams can source data, train models, monitor model health, and deploy recommendations into the field workflow.

Together, these modules form a closed-loop system where field execution outcomes can continuously improve future recommendations.

## Key Capabilities

- Personalised weekly NBA journey plans for HCPs
- Tactic tracking across email, calls, CLM sessions, and 1:1 meetings
- HCP-level performance and engagement visibility
- Territory heat maps and responsiveness insights
- No-code model creation and deployment
- Built-in model monitoring and governance
- Integration-ready architecture for CRM, ERP, DWH, and BI platforms

## Target Users

- Field Medical Representatives
- Commercial Analytics / Data Science Teams
- Commercial Excellence Leaders
- Marketing Operations / Brand Teams

## Product Vision

Pharma Action Builder is designed to become the intelligence layer between commercial analytics and field execution — helping pharma organizations operationalize data science faster and drive more relevant HCP engagement.
